推荐解决方案（强烈建议）：使用虚拟环境（venv）这是 Python 开发的最佳实践，以后每次做项目都应该这么做。你在 week1 文件夹里，执行下面4条命令（一行一行复制粘贴）：bash

# 1. 创建一个叫 venv 的虚拟环境
python3 -m venv venv

# 2. 激活虚拟环境（激活后你的终端提示符会变）
source venv/bin/activate

# 3. 在虚拟环境里安装 requests
python3 -m pip install requests

# 4. 验证安装成功
python3 -c "import requests; print('✅ requests 安装成功！版本：', requests.__version__)"

安装成功后，你就可以正常运行你的脚本了：bash

python3 api_requests.py

以后每次使用这个项目时记得激活环境bash

cd ~/ai-learning/week1          # 进入你的项目文件夹
source venv/bin/activate        # 激活（提示符会变成 (venv) 开头）
python3 api_requests.py         # 运行脚本

退出虚拟环境可以用：bash

deactivate
